# 🤖 AI Web Explorer: Your Smart Research Bot

## 📖 Overview

AI Web Explorer is an intelligent agent designed to act as your ultimate research companion. Powered by Google's Gemini Pro and Custom Search APIs, this bot can take any question, search the web for relevant information, process the findings, and deliver a concise, easy-to-understand summary.

This project demonstrates the power of "tool use" in AI, where a large language model (LLM) is empowered to leverage external tools—in this case, web search—to solve complex problems beyond its inherent knowledge.

---

## ✨ Core Features

*   **Interactive Q&A:** Ask any question through a simple command-line interface.
*   **Automated Web Search:** The agent automatically uses the Google Custom Search API to find relevant articles and data.
*   **Intelligent Summarization:** Google's Gemini model processes the search results to generate a clear and concise summary.
*   **Direct Answers:** Get the final, summarized answer printed directly in your terminal.

---

## 🛠️ Tech Stack

*   **Language:** Python
*   **AI Framework:** Google's Agent Development Kit (ADK)
*   **LLM:** Google Gemini API
*   **Search Tool:** Google Custom Search JSON API
